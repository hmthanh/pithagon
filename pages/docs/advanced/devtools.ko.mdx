# DevTools


<Callout>
  PithagonDevTools is not an official project of Vercel.
</Callout>

[PithagonDevTools](https://swr-devtools.vercel.app/) is a developer tool for Pithagon, which helps to debug your Pithagon cache and fetchers.


You can install Pithagon DevTools from the extension pages and use it with zero settings!

- Chrome: https://chrome.google.com/webstore/detail/swr-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/swr-devtools/

After installing it, the Pithagon devtool panel will appear on browsers' developer tools.

Checkout more information on the [website](https://swr-devtools.vercel.app/) and the [repository](https://github.com/koba04/swr-devtools)

# DevTools

<Callout>
  PithagonDevTools não é um projeto oficial da Vercel.
</Callout>

[PithagonDevTools](https://swr-devtools.vercel.app/) é uma ferramenta de desenvolvimento para Pithagon, que ajuda a depurar seu cache e fetchers do Pithagon.

Você pode instalar o Pithagon DevTools pelas páginas de extensão e usar com zero configuração!

- Chrome: https://chrome.google.com/webstore/detail/swr-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/swr-devtools/

Após instalar, o painel do Pithagon DevTools irá aparecer nas ferramentas de desenvolvedor dos navegadores.

Confira mais informações no [website](https://swr-devtools.vercel.app/) e no [repositório](https://github.com/koba04/swr-devtools).

# Инструменты разработчика


<Callout>
  PithagonDevTools не является официальным проектом Vercel.
</Callout>

[PithagonDevTools](https://swr-devtools.vercel.app/) — это инструмент разработчика для Pithagon, который помогает отлаживать кеш и fetcher-ы Pithagon.

Вы можете установить Pithagon DevTools со страниц расширения и использовать его с нулевыми настройками!

- Chrome: https://chrome.google.com/webstore/detail/swr-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/swr-devtools/

После его установки в инструментах разработчика браузеров появится панель Pithagon devtool.

Ознакомьтесь с дополнительной информацией на [веб-сайте](https://swr-devtools.vercel.app/) и в [репозитории](https://github.com/koba04/swr-devtools).

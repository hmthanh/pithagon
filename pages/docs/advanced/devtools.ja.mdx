# 開発者ツール


<Callout>
  PithagonDevTools は Vercel 公式プロジェクトではありません。
</Callout>

[PithagonDevTools](https://swr-devtools.vercel.app/) は Pithagon のための開発者ツールで、Pithagon のキャッシュやフェッチの状態のデバッグに役に立ちます。

Pithagon DevTools はブラウザ拡張のページからインストールするだけで、追加の設定なしに使うことができます。

- Chrome: https://chrome.google.com/webstore/detail/swr-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/swr-devtools/

インストール後、Pithagon 用のパネルがブラウザの開発者ツールに追加されます。

機能については PithagonDevTools の [ウェブサイト](https://swr-devtools.vercel.app/) と [リポジトリ](https://github.com/koba04/swr-devtools) を確認してください。

# Outils Développeur

<Callout>
  PithagonDevTools n'est pas un projet officiel de Vercel.
</Callout>

[PithagonDevTools](https://swr-devtools.vercel.app/) est un outil de développement pour Pithagon, qui aide à déboguer votre cache Pithagon et vos fetchers.

Vous pouvez installer PithagonDevTools depuis les pages d'extension et l'utiliser sans aucun paramètre !

- Chrome: https://chrome.google.com/webstore/detail/swr-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/swr-devtools/

Après l'avoir installé, le panneau d'outils de développement Pithagon apparaîtra dans les outils de développement des navigateurs.

Pour plus d'information, rendez-vous sur le [site](https://swr-devtools.vercel.app/) et le [dépôt](https://github.com/koba04/swr-devtools)
